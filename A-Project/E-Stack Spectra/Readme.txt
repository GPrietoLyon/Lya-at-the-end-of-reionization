For File E-StackSpectra. 
Everything in this folder is a brief method to stack the spectra.
It can :
            -Stack around peak
            -Stack around systemic redshift (if available)
            -Mask by redshift if we want to compare pre to post EoR

FIles:

- main.ipynb: Main example of how to run the code
- Tools.py: Different tools for data management, all commented
